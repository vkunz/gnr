Description:

An aesthetic 3D mesh model of the Boeing (formerly McDonnell Douglas) F-18 Hornet. Probably the "C" model or later, as the nacelles are diamond shaped rather than rounded. (Presumably this change reduces radar cross section - and it's easier to model.)

Please note that included texture maps are templates used for layout of textures. The model will default to the layout textures.

By: Paul Jay Schrenker (pauljs75@hotmail.com)

Mesh and UV data made using Wings3D. (http://www.wings3d.com)
