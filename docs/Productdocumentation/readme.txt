Um das Programm zu installieren entpacken Sie bitte das selbstentpackende Archiv in ein frei wählbares Verzeichnis. Darin enthalten sind die Programmdatei, eine Beispiel-Library, sowie eine HTML-Hilfe.
Nach dem Entpacken starten Sie GNR.exe um die Anwendung zu starten.

Immer, wenn Sie während der Programmausführung Hilfe benötigen drücken Sie F1 um eine HTML-Hilfe anzuzeigen.
